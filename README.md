📂 Dataset
The dataset used in this project is too large to be stored directly in this repository.
You can download it from Kaggle:
🔗 https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
📥 Steps to Use the Dataset
Download creditcard.csv from the Kaggle link above.
Create a folder named data in the root directory of this project (if it does not already exist).
Place the downloaded file inside the data folder:

project-root/
│
├── data/
│   └── creditcard.csv
│
├── notebooks/
├── src/
└── README.md
